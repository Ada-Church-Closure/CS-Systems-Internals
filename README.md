这是我本人做的一些关于名校公开课或者是收集的高质量的课程资料仓库。

------

- 1.[CMU CS15213: CSAPP](https://csapp.cs.cmu.edu)	(2025.3-2025.6)

  > ​	这是学校开设的ICS的课程.并非我自学,而且过于著名,也不需要我的评价,对于system感兴趣很适合从这里开始入手.

- 2.[CS61A: Structure and Interpretation of Computer Programs](https://insideempire.github.io/CS61A-Website-Archive/)	(2025.6-2025.8)

  > ​	一共用时大约两个月左右,因为没有集中的大量时间,中间还穿插了七八个考试,在2025年8月上旬结束,最有意思的项目是scheme解释器,所有project对我来说难度都不大,理解难度大于编码,并且基本没有很硬核的debug之类的内容,适合入门并且扩展见识,但是第四章的内容没有认真看,对了,主要还是py.
  >
  > ​	上面的link是2024年版本的存档,你还可以在学习中参考这个[中文手册](https://composingprograms.netlify.app),我也参与了简单的订正.
  
- 3.[MIT 6.S081 Operating System Engineering](https://pdos.csail.mit.edu/6.828/2021/schedule.html)	(2025.9-2025.11)

> ​	基本完成90%,主要看一些全课程翻译网站和总结,跟着做lab,代码量整体较少,但是很吃对于OS的理解,而且自己debug的话可能会相当困难,课程涉及的几篇论文和剩下的内容还会抽时间再看.
